# Announcement

`Announcement` - модуль объявлений в верхней части сайта. Он показывает важные сообщения всем посетителям: новости, предупреждения, акции и обновления.

**Что умеет:**

* Разные стили объявлений: инфо, успех, предупреждение, ошибка
* Можно добавить кнопку “Подробнее” со ссылкой
* Объявления можно включать/выключать и менять порядок
* Можно настроить показ по времени (с/до)
* Можно сделать объявление закрываемым для пользователя

---

`Announcement` is a top-of-site announcements module. It displays important messages for visitors such as news, updates, warnings, or promotions.

**What it offers:**

* Multiple announcement styles: info, success, warning, error
* Optional action button with a link
* Enable/disable announcements and change their order
* Schedule visibility (show from / until)
* Dismissible announcements that users can close

## Installation

Download the latest release and install it via the Flute CMS admin panel.

Current version: **2.0.0**
