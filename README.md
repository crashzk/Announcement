# Announcement
`Announcement` - модуль объявлений в верхней части сайта. Он показывает важные сообщения всем посетителям: новости, предупреждения, акции и обновления.  **Что умеет:**  * Разные стили объявлений: инфо, успех, предупреждение, ошибка * Можно добавить кнопку “Подробнее” со ссылкой * Объявления можно включать/выключать и менять порядок * Можно настроит
